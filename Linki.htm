Linki

Jak to już tłumaczyłem, strasznie irytowało mnie to, że linki były podkreślone automatycznie i podczas szukania
jak się tego pozbyć, odkryłem też, że można zmieniać takie rzeczy jak to, iż link po kliknięciu nie zmienia koloru,
a także ustawic inne rzeczy!

Komenda dla braku podkreśleń w CSS "a { text-decoration: none; }"

A tu pokaże fragment tekstu do różnych zmian oraz przykladowe linki, by pokazać jak ja to zrobiłem.
"/* nieodwiedzony link */
a:link {
    color: red;
}

/* odwiedzony link */
a:visited {
    color: green;
}

/* gdy myszka nad linkiem */
a:hover {
    color: hotpink;
}

/* wybrany link */
a:active {
    color: blue;
}"
